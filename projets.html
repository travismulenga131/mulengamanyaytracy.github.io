Projet 1: Site E-commerce - FashionStore

Le projet "FashionStore" est un site e-commerce complet, conçu pour la vente en ligne de vêtements. J'ai utilisé HTML, CSS, JavaScript et PHP pour créer une expérience utilisateur fluide. Ce site inclut une page d'accueil dynamique, un système de gestion de panier, et une intégration sécurisée des paiements via Stripe. J'ai également mis en place un tableau de bord pour l'administration des produits, des commandes et des utilisateurs.

Technologies utilisées : HTML, CSS, JavaScript, PHP, MySQL, Stripe API

Projet 2: Application Mobile - FitnessApp

Ce projet "FitnessApp" est une application mobile de suivi des activités sportives, permettant aux utilisateurs de suivre leur progression, de définir des objectifs et de consulter des conseils personnalisés.
